# 深度定制化 UI 爆改方案 (UI Design Plan)

## 核心设计理念：从“通用聊天框”到“沉浸式写作 IDE”
Open-WebUI 的原始界面是完全照搬 ChatGPT 的“问答式”布局，这对于普通闲聊足够，但对于专业网文创作者来说，简直就是“灾难”——功能入口极其琐碎（一堆对码字毫无用处的极客配置），且完全没有“主工作区”的概念。

我们的目标是：做**减法**去除冗余，做**加法**升维颜值，将它彻底改造成一款**暗黑/轻奢风格的高端 AI 写作引擎**。

---

## 🔪 第一步：极致的“减法” (Feature Culling)
网文作者不需要关心底层架构，我们需要把原始 UI 中所有干扰创作心流的“极客功能”全部斩断（通过隐藏代码或 CSS `display: none`）：

1. **左侧边栏净化**：
   - 砍掉 `Workspace` (知识库/提示词库)。
   - 砍掉 `Prompts`、`Modelfiles` 菜单。
   - 砍掉底部多余的用户管理、权限分配等 Admin 面板入口。
   - **只保留**：新建分支（章节）、历史分支列表。
2. **主聊天区净化**：
   - 砍掉输入框旁边的“语音输入(TTS)”、“上传图片(Vision)”、“通话(Call)”等分散注意力的按钮。
   - 砍掉多模型对比 (Arena) 和复杂的模型选择器（因为我们已经锁死了 Gemma-4-31B，模型选择器甚至可以完全隐藏，只留一个优雅的引擎状态灯）。
3. **设置面板净化**：
   - 将“通用/连接/语音/图片/管理员”等几十个设置项彻底折叠或屏蔽，只留下我们专用的“高级对话设置（参数调节）”。

---

## 🏗️ 第二步：空间架构重组 (Layout Restructuring)
打破 ChatGPT 的单列居中布局，改为**专业级三栏式/双栏式写作布局**：

### 1. 左侧：章节树状栏 (Chapter Navigator)
- 将原本的“对话记录”改造成类似 Scrivener / Notion 的“章节目录”。
- 增加悬停效果，支持文件夹形式归档不同的卷章。

### 2. 中间主视觉：沉浸式创作视窗 (Writing Canvas)
- **拓宽视野**：打破原来 `max-w-3xl` 的窄条居中限制，将对话视窗拓宽，给小说文本留出更舒展的阅读空间。
- **对话框改造**：人类发送的提示词（比如“写下一章，重点描写战斗”）可以缩小、弱化显示；而大模型返回的**小说正文**，则以大字号、精美排版的“无边框纸张”形式呈现，弱化“气泡框”的割裂感。
- **底部输入台**：输入框不再是一个死板的框，而是悬浮式的毛玻璃 (Glassmorphism) 控制台。

### 3. 右侧：可折叠的控制中枢 (The Command Center)
目前我们把所有的神器（防爆阀、仪表盘、记忆面板）都塞在了顶部导航栏，这其实很不优雅。
- **方案**：引入一个右侧边栏（按 `Tab` 或按钮可弹出/隐藏）。
- **内容**：
  - **模块 A (仪表盘)**：Context Load 能量条和字数监控，以更直观的圆环或进度条显示。
  - **模块 B (卡片插拔区)**：直接将 Memory Vault（记忆卡片）平铺在右侧，像卡牌游戏一样，作者可以边看正文，边在右侧快速点亮/熄灭某些设定卡片。
  - **模块 C (幽灵总结器)**：随时提取当前情节，直接在右边栏生成大纲。

---

## 🎨 第三步：颜值与质感升维 (Aesthetic Upgrades)
我们要让用户打开这个软件的第一眼，就感受到一种“极其专业、昂贵且沉稳”的氛围。

1. **色彩哲学 (Dark & Accent)**：
   - **主色调**：深邃的黑灰色（如 `#0F1115` 或 `#121212`），极度护眼。
   - **点缀色 (Accent)**：摒弃原始的亮蓝色，改用极具高级感的**暗金/琥珀色 (Amber/Gold)** 或**幽灵青 (Cyan/Teal)** 作为高亮和按钮色。
   - **状态色**：保留您设计的防爆阀（粉红色）和仪表盘警报色。
2. **排版与字体 (Typography)**：
   - **系统 UI**：采用现代化的无衬线字体（如 Inter / Roboto）。
   - **小说正文**：强制引入更具文学气息的**衬线体**（如 Noto Serif SC / 思源宋体），增大行距 (Leading) 和段间距，让网文阅读体验媲美实体书排版。
3. **质感微交互 (Micro-Animations)**：
   - 所有的切换、弹出、卸载卡片，必须带有丝滑的非线性动画（使用 Svelte 的 `fly` / `fade` 配合弹簧缓动曲线）。
   - 输入框和悬浮面板使用毛玻璃特效（`backdrop-blur-xl` + 极其微弱的白色/金色边框）。

## 🗣️ 您的想法？
这相当于我们要把前端组件从骨架到皮肤全都扒下来重写。
您可以先看看这个大方向：
1. **减法**部分，您觉得还有哪些平时绝对用不到的功能，我想办法直接给它“物理超度”？
2. **右侧控制中枢**的设计，您觉得会不会比现在全部挤在顶部更好操作？
3. **色彩和字体**，您更倾向于“高级极简风”还是带有某种特定氛围（如赛博朋克、古风或冷峻工业风）的配色？
